# Grok Be Gone

A browser extension for **X** (formerly Twitter) that strips Grok-related UI from the site: sidebar entries, in-app panels, post actions, and optional filtering of posts that mention or link to Grok. Settings are synced across devices and work the same in Chromium-based browsers and Firefox.

## Highlights

- Hide Grok in the sidebar, drawer, profile cards, and post composer extras (Enhance / Explain).
- Optionally hide timeline posts that tag **@grok** or link to Grok.
- Declutter the sidebar and overflow menu (Communities, Premium, ads, jobs, and more) with granular toggles.
- Optional removal of the Premium subscribe promo card.
- Early application of styles so clutter is hidden as soon as possible on page load.
